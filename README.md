# Wei Xie - Fullstack Engineer

This is a personal website for Wei Xie, built with Vue 3, TypeScript, Tailwind CSS, and Vite.
The design focuses on a clean, professional, and minimal aesthetic suitable for a senior engineer.

## Architecture

- **Frontend**: Vue 3 + TypeScript
- **Styling**: Tailwind CSS
- **Internationalization**: Vue I18n (English & French)
- **Build Tool**: Vite
- **Deployment**: GitHub Pages

## Project Setup

### Install Dependencies

```sh
npm install
```

### Development Server

Start the development server:

```sh
npm run dev
```

### Production Build

Build the application for production:

```sh
npm run build
```

The build artifacts will be stored in the `dist/` directory.

## Features

- **Responsive Design**: Mobile-first approach using Tailwind CSS.
- **Bilingual**: Switch between English and French.
- **Clean Architecture**: Component-based structure with reusable wrappers.

## Structure

- `src/components`: Contains all section components and UI elements.
- `src/i18n`: Locale definitions (en.json, fr.json).
- `src/style.css`: Tailwind directives and custom base styles.
- `src/App.vue`: Main application entry point.
